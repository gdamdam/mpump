# Genre Mix Profiles — Actionable Mixing Reference

Research compiled from: Sound on Sound, Attack Magazine, iZotope, MusicRadar, DJ TechTools (2024–2026).

---

## How to read this table

| Column | What it means |
|--------|---------------|
| **Kick / Bass / Synth** | Relative volume offsets from a nominal 0 dB fader. Kick is usually the loudest element in dance music; these are rough starting targets. |
| **Sidechain duck** | Amount of bass gain reduction when kick fires. ~3 dB = subtle/transparent. 6–10 dB = audible pump. 15–25 dB = aggressive/extreme. |
| **Key EQ moves** | Most genre-defining frequency decisions. Sub = <80 Hz, Low-mid = 150–400 Hz, Hi-mid = 1–5 kHz, Air = 8–16 kHz. |
| **Stereo width** | Bass/kick always mono (or sum <120 Hz to mono). Width refers to synth/pad/top-end treatment. |

## 1. Techno (warehouse / Berghain style)

| Parameter | Value |
|-----------|-------|
| **Kick level** | Reference (0 dBFS headroom target) |
| **Bass vs kick** | Bass sub layer LP-filtered at ~80 Hz; bassline body 98 Hz–1 kHz; kick dominates <100 Hz |
| **Synth vs kick** | –3 to –6 dB below kick at fader |
| **Sidechain duck** | Moderate-strong: 6–10 dB gain reduction on bass, sub sidechained via transient shaper (attack pulled down so sub ducks on its own transient) |
| **EQ: kick** | Boost 70–90 Hz for thump; cut 180 Hz (boxiness); +2–3 dB at 3–4 kHz for click/attack |
| **EQ: bass** | Sub layer LP at 80 Hz (12 dB/oct); bassline HP around 90–100 Hz; boost 220–300 Hz for mid-range body on warehouse systems; cut 500 Hz–1 kHz (mud) |
| **EQ: synth** | Hard HP at 150–200 Hz (keep lows clear); boost 3–5 kHz for presence on loud systems |
| **Stereo width** | Bass and kick strictly mono; percussive top layer widened with chorus/stereo plugin; synth leads narrow to mid-wide |

**Character notes:** Multiple strata of clearly separated low end are the hallmark — sub → bassline → percussive layer all occupy different frequency bands. Kick must punch through heavy ambient textures so sidechain on chords/pads is essential.

Sources: [Attack Magazine — Techno Basslines](https://www.attackmagazine.com/technique/tutorials/warehouse-rolling-techno-bass/), [Attack Magazine — Dub Techno Beat](https://www.attackmagazine.com/technique/beat-dissected/basic-channel-style-dub-techno/)

## 2. House / Deep House

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick wins the low-end war; bass pushed to start at ~100 Hz |
| **Bass vs kick** | –3 dB at fader; bass sits above kick's dominant frequency |
| **Synth/chord vs kick** | –6 dB or more; heavily sidechained during drop |
| **Sidechain duck** | Audible pump: 6–8 dB during drops; more subtle (3 dB) in deep house grooves |
| **EQ: kick** | Boost 70–90 Hz; cut 180 Hz; boost 3–4 kHz for click; HP below 30–40 Hz |
| **EQ: bass** | Boost 150–500 Hz ("warm, funky" low-mid); cut at kick's fundamental to leave space |
| **EQ: synth** | Warm low-mid (200–400 Hz boost for richness in deep house); no aggressive high-shelf |
| **Stereo width** | Bass mono below 120 Hz; Rhodes/chord stabs moderate width; pads wide |

**Character notes:** The four-on-the-floor kick is the heart of the genre. Deep house favors warmth in the 150–500 Hz range vs. techno's sparse low-mid. Sidechain should feel groovy, not jarring.

Sources: [iZotope — 9 Tips for House Music](https://www.izotope.com/en/learn/9-tips-for-producing-and-mixing-house-music.html), [Sound on Sound — Secrets of House & Trance](https://www.soundonsound.com/techniques/secrets-house-trance)

## 3. Trance / Psytrance

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick prominent; bass acts as rhythmic energy, often in higher register |
| **Bass vs kick** | –3 to –4 dB; bass often starts at 100 Hz+ |
| **Synth vs kick** | Supersaw/lead at –4 to –6 dB; pad layers lower |
| **Sidechain duck** | Strong pumping on synths/pads: 8–12 dB GR; bass duck 6 dB |
| **EQ: kick** | Boost 70–90 Hz; cut 150–180 Hz; +3 dB at 3–4 kHz for snap |
| **EQ: bass** | Boost ~1 kHz and 3 kHz to add "presence" — makes the bass cut even at high volumes; sub HP of kick sound to carve low space for bass |
| **EQ: synth** | Air boost at 8–13 kHz (+2–4 dB) gives the characteristic trance "shine"; boost 1–3 kHz for presence |
| **Stereo width** | Bass mono; supersaw leads very wide (stereo unison); pads maximum width |

**Character notes:** Trance is one of the few genres where boosting treble/air EQ on bass sounds right (13 kHz and 3 kHz on bassline, per MusicRadar trance tips). Synth supersaws are typically very wide. Pumping sidechain on pads is expected and loud.

Sources: [MusicRadar — 26 Trance Tips](https://www.musicradar.com/tuition/tech/26-trance-tips-34006), [Sound on Sound — Secrets of House & Trance](https://www.soundonsound.com/techniques/secrets-house-trance)

## 4. Drum & Bass / Jungle

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick punchy but bass/sub carries equal weight in dnb |
| **Bass vs kick** | 0 to –2 dB; bass is co-dominant |
| **Synth vs kick** | –6 to –9 dB; pads/stabs heavily ducked |
| **Sidechain duck** | Moderate: 4–6 dB; fast attack (≤1 ms), release tuned to beat (≈30–80 ms) |
| **EQ: kick** | Boost 80 Hz for sub punch; boost 2–4 kHz for snap; cut 150–200 Hz (mud) |
| **EQ: bass** | Reese/growl bass occupies 80–500 Hz heavily; cut kick's fundamental range on bass; boosting 220–300 Hz adds character |
| **EQ: breaks** | Narrow-band cuts in problem areas + harmonic exciter on top end (rather than flat EQ boosts); layer TR-909 under breaks for definition |
| **Stereo width** | Sub bass mono; Reese/neuro bass can be wide (unison detune); breaks can be wide with mid-side processing |

**Character notes:** Jungle specifically uses sampled breaks + sub bass — the break is the centerpiece, not a 4-on-the-floor kick. DnB uses syncopated programmed kicks. Both require a strong sub-to-high-end contrast ("massive sub + crisp top"). Parallel compression on break (8:1+, 25 dB GR) gives the squashed/pumped character.

Sources: [Attack Magazine — Mixing Breakbeats](https://www.attackmagazine.com/technique/tutorials/mixing-breakbeats/), [iZotope — Mix Kick and Bass](https://www.izotope.com/en/learn/how-to-mix-kick-and-bass)

## 5. Dubstep

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick focuses on 80–120 Hz punch; sub bass lives at 30–60 Hz (C0–C1) |
| **Bass vs kick** | Sub bass dominant during bass sections; kick stays out of sub range |
| **Synth vs kick** | Wobble bass IS the lead; –3 to –6 dB vs kick at drop |
| **Sidechain duck** | Strong: 8–12 dB on sub when kick fires; use multiband sidechain (duck only 0–200 Hz) for cleaner result |
| **EQ: kick** | Avoid 808-style kick (muddy with sub); use punchier kick at 80–100 Hz; HP bass kick below 80 Hz |
| **EQ: sub bass** | Sine-based, LP at 60 Hz; kick sits above sub at 80–100 Hz |
| **EQ: wobble bass** | Mid-bass 100–500 Hz; LFO filter sweep is the sound — resonant LP or BP |
| **Stereo width** | Sub bass strictly mono; wobble can be moderate width; reverb snares are wide |

**Character notes:** Half-time drum programming (snare on beat 3 only) defines the genre. Sub bass and kick must occupy separate frequency bands — sub at 30–60 Hz, kick body at 80–120 Hz. Reverberated layered snares are essential.

Sources: [Sound on Sound — Dubstep Basics](https://www.soundonsound.com/techniques/dubstep-basics), [iZotope — Mix 808s](https://www.izotope.com/en/learn/how-to-mix-808s)

## 6. Ambient / Dub Techno

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick is supportive, not dominant; Basic Channel style = kick through heavy reverb |
| **Bass vs kick** | Bass and kick roughly equal; both treated with space/texture |
| **Synth/chord vs kick** | Dub chords at –6 dB but sit in a wide space |
| **Sidechain duck** | Subtle: 2–4 dB on ambient textures; sidechain on chords/pads is essential to let kick punch through reverb |
| **EQ: kick** | Cut harsh highs; preserve sub and low-mid warmth; EQ cutoff filter automation common |
| **EQ: bass** | Sub-heavy, warm; HP on anything above 500 Hz; cut noise from distortion with high-shelf |
| **EQ: chords/pads** | BP filter with resonance for dub character; low-shelf cut at 450 Hz (HP at 450 with 12 dB/oct slope); automate filter cutoff |
| **Stereo width** | Mono bass core; everything else pushed very wide with delays and reverbs; mix of mono + stereo reverb for depth |

**Character notes:** Tape delays (1/8 dotted), filter delays, and reverb (long decay, high damping) are the defining sound. Filtering bandpass with cutoff automation on chord stabs is how Basic Channel-style texture is made. Keep bass mono — all width comes from delays/reverbs on mid and high content.

Sources: [Attack Magazine — Basic Channel Dub Techno](https://www.attackmagazine.com/technique/beat-dissected/basic-channel-style-dub-techno/), [Waves — Dub Mixing Tips](https://www.waves.com/dub-mixing-tips-daniel-boyle)

## 7. Acid Techno

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick dominant, acid line is mid-frequency lead |
| **Bass vs kick** | –3 dB; 303 acid line occupies 100 Hz–2 kHz (not deep sub) |
| **Synth vs kick** | Acid line IS the synth; –3 to –4 dB |
| **Sidechain duck** | Strong: 6–10 dB; acid line pumps with kick for driving effect |
| **EQ: kick** | Standard techno kick EQ (80 Hz boost, 180 Hz cut, 3–4 kHz click) |
| **EQ: acid line** | 303-style filter sweep does the EQ work; cut below 100 Hz to keep sub clear; resonant filter peak at cutoff gives characteristic mid boost |
| **Stereo width** | Mono bass; acid line mono or slight stereo chorus for width |

**Character notes:** The acid 303 line occupies the mid-range, not the sub. This means less low-end competition with the kick vs. other genres — but the resonant filter peak can clash with kick's 3–4 kHz click. Automate the filter cutoff; that IS the mix movement.

Sources: [Attack Magazine — Techno Basslines](https://www.attackmagazine.com/technique/tutorials/warehouse-rolling-techno-bass/), [Sound on Sound — Secrets of House & Trance](https://www.soundonsound.com/techniques/secrets-house-trance)

## 8. IDM / Glitch

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | No fixed four-on-the-floor; kick is a sound design element, often irregular |
| **Bass vs kick** | Bass at equal level or lower than the kick; both are timbral |
| **Synth vs kick** | Textural synths at –6 dB; lots of detail in mid-high range |
| **Sidechain duck** | Subtle to none for ambient IDM; moderate (4–6 dB) for rhythmic IDM tracks |
| **EQ: kick** | Bitcrushed/processed kicks — EQ for interest, not genre rule; cut mud at 200–400 Hz |
| **EQ: bass** | Distorted/resampled bass; keep sub mono; cut 200–400 Hz mud; use saturation for harmonics |
| **EQ: glitch elements** | Bitcrushing + sample-rate reduction creates its own spectral character; no standard EQ moves |
| **Stereo width** | Very wide on textural elements; bass mono; glitched loops can be hard panned |

**Character notes:** IDM/glitch rejects genre conventions. The "rule" is no rule. Sub bass stays mono. Processing chain is more important than mix level relationships. Interesting timbres > loud levels.

## 9. Electro / Breakbeat

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Electro: kick dominant (808 body kick); Breakbeat: break is central, kick layered under |
| **Bass vs kick** | –2 to –4 dB; bass must stay clear of break samples |
| **Synth vs kick** | –4 to –6 dB; electro leads are cutting and present (saw/sync) |
| **Sidechain duck** | Moderate: 4–6 dB; breakbeat tracks use parallel compression on breaks (8:1+) for squashed energy |
| **EQ: kick** | Electro: 808 body at 50–80 Hz; boost 3–4 kHz for click. Breakbeat: 80 Hz boost on kick layered under break |
| **EQ: breaks** | Narrow cuts for resonances; harmonic exciter at 3–8 kHz for clarity (better than broad EQ boosts); layer 909 under break for definition |
| **EQ: synth** | Hard HP at 200 Hz; boost 2–5 kHz for electro "robot" sound; cut 800 Hz (nasality) |
| **Stereo width** | Electro: bass mono, leads narrow; Breakbeat: wide stereo break (M-S processing); synths hard-panned |

**Character notes:** Breakbeats require subtlety with EQ — pushing low-mid on a break results in mud. Use exciter/enhancer above 3 kHz instead. 909 samples layered subtly add more clarity than any EQ. For electro, the 808 body kick is the instrument — don't EQ away the 50–80 Hz.

Sources: [Attack Magazine — Mixing Breakbeats](https://www.attackmagazine.com/technique/tutorials/mixing-breakbeats/), [MusicRadar — Breakbeat Production](https://www.musicradar.com/tuition/tech/7-breakbeat-production-problems-solved-197911)

## 10. Lo-fi / Downtempo

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Lo-fi: equal; both are soft, warm, vintage-sounding |
| **Bass vs kick** | 0 to –2 dB; bass is warm and present |
| **Synth vs kick** | –3 to –6 dB; texture over presence |
| **Sidechain duck** | Subtle or none: 0–3 dB; lo-fi avoids pumping |
| **EQ: kick** | Soft; LP at 8–12 kHz; boost 80 Hz for weight; cut 3–5 kHz (less click = less hi-fi) |
| **EQ: bass** | Tape-style: boost at 80–100 Hz ("tape bump"); LP at 8–10 kHz; cut 300–400 Hz (mud) |
| **EQ: synth/texture** | HP at 100 Hz; LP at 8–12 kHz; boost 1–4 kHz for presence where high-end is cut |
| **Stereo width** | Fold pads to mono or narrow for vintage feel; pull leads center; sum reverb to mono for old-school depth |

**Character notes:** The defining move is high-frequency roll-off: LP at ~12 kHz (or lower for extreme lo-fi) combined with tape saturation. Drive a tape emulator at 7.5 IPS for "bass-forward with high-freq loss." Push small EQ bump at 1–4 kHz to restore mix presence without adding brightness.

Sources: [MusicRadar — Lo-fi Processing Tips](https://www.musicradar.com/how-to/lo-fi-processing-tps), [Attack Magazine — Lo-fi Sound](https://www.attackmagazine.com/technique/tutorials/the-secrets-of-dance-music-production-lo-fi-sound/), [Sound on Sound — Lo-fi Beats](https://www.soundonsound.com/techniques/sos-artists-lo-fi-beats-modern-plug-ins)

## 11. Synthwave

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick prominent but warm; bass is melodic and present |
| **Bass vs kick** | –2 to –3 dB; bass is prominent and melodic |
| **Synth vs kick** | –3 to –5 dB; pads/leads are the star |
| **Sidechain duck** | Subtle to moderate: 3–6 dB; slight pumping gives 80s feel |
| **EQ: kick** | Vintage feel: boost 80 Hz; gentle LP at 10 kHz; cut 500–800 Hz (boxy) |
| **EQ: bass** | Boost 200–400 Hz for warmth; gentle HP at 40 Hz; tape saturation for harmonics |
| **EQ: synth pads** | Boost 8–12 kHz (air); boost 1–2 kHz for retro presence; gentle HP at 150 Hz |
| **Stereo width** | Bass mono or narrow; chorus on pads (very wide); leads moderate width; drums punchy center |

**Character notes:** Synthwave is a lo-fi-adjacent genre sonically — analog warmth, tape-style character, but with prominent leads. Unlike strict lo-fi it retains treble on synths. Chorus (3-voice) on pads creates the signature wide, lush sound. Avoid over-compressed, clinical mixes — leave some dynamics.

## 12. Garage / 2-Step

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick is syncopated (2-step); bass/sub is driving, swung |
| **Bass vs kick** | Bass 0 to –2 dB; bass groove is central |
| **Synth/vocal vs kick** | –4 to –6 dB |
| **Sidechain duck** | Subtle: 2–4 dB; garage avoids heavy pumping (that's house/trance territory) |
| **EQ: kick** | Crisp; boost 80 Hz; boost 3–5 kHz for click; cut 200–400 Hz (hollow) |
| **EQ: bass** | Swung sub; boost 60–80 Hz; cut 200–400 Hz (mud); bass often pitched/melodic |
| **EQ: synth/vocal** | Boost 2–5 kHz for presence; gentle HP at 200 Hz |
| **Stereo width** | Sub mono; bass can have slight width; vocals and strings wide |

**Character notes:** Two-step swing pattern displaces the kick — the off-beat feel is the genre. Sub bass is swung with the beat pattern, not static. UK garage often includes pitched sub bass following vocal melodies. Avoid excessive sidechain pump — it fights the groove.

Sources: [Sound on Sound — Dubstep Basics](https://www.soundonsound.com/techniques/dubstep-basics) (garage roots context)

## 13. EDM (festival / commercial)

| Parameter | Value |
|-----------|-------|
| **Kick vs bass** | Kick is aggressive and dominant; full-spectrum kick design |
| **Bass vs kick** | –3 to –4 dB; bass heavy, often 808-style |
| **Synth/lead vs kick** | –4 to –6 dB pre-drop; heavily sidechained during drop |
| **Sidechain duck** | Very strong: 10–15 dB on synths during drop; visible "breathing" pumping |
| **EQ: kick** | Boost 50–80 Hz; boost 2–5 kHz for transient; cut 120–200 Hz (mud); HP below 30 Hz |
| **EQ: bass** | HP cut below kick's fundamental; boost 80–100 Hz; cut 200–400 Hz; 808s: gentle HP at 40 Hz, body at 60–80 Hz |
| **EQ: lead synth** | HP at 200 Hz; boost 2–5 kHz for "big room" cut-through; high shelf boost 8–12 kHz |
| **Stereo width** | Bass and 808 mono below 120 Hz; everything else maximum width; supersaws wide; reverb tails wide |

**Character notes:** EDM drops require extreme sidechain pump — it's an expected sonic feature. Mix EDM with the master limiter in mind: low-end headroom is the main constraint. Reference master level while adjusting kick+bass to check headroom recovery (per Stimming's DJTechTools tip).

Sources: [iZotope — 13 Tips for Mixing EDM](https://www.izotope.com/en/learn/12-tips-for-mixing-and-producing-edm.html), [DJ TechTools — 5 Kick+Bass Secrets](https://djtechtools.com/2016/10/21/5-kick-bass-mixdown-secrets-top-producers/)

## Quick-reference sidechain settings

| Strength | Gain Reduction | Ratio | Use case |
|----------|---------------|-------|----------|
| Subtle | 2–3 dB | 2:1 | Lo-fi, ambient, garage |
| Moderate | 4–6 dB | 3:1–4:1 | House, DnB, dub techno |
| Strong | 6–10 dB | 4:1–8:1 | Techno, acid, dubstep |
| Extreme pump | 10–25 dB | 8:1–inf | EDM drops, trance drops, hard house |

Attack: always fast (≤1 ms) so kick punches through. Release: tune by ear to match beat rhythm — start at maximum pump, back down.

## Quick-reference: mono bass rule

Everything below ~120 Hz should be mono (or summed to mono via M-S processing) in all genres except deliberate stereo-bass experiments. This preserves low-end power on mono playback systems (clubs, phones, Bluetooth speakers).
