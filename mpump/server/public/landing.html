core
// how it works
You make a beat. It becomes a link.
Send it to a friend. They hear exactly what you hear.
They change it. They send it back different.

The link carries everything. Open it anywhere. Change it.
Send it back different.

start
// starting points
Not from silence.
From something that already has a pulse.

Every session opens with a working groove. Everything is already moving. You decide where it goes next.

diff
// not a DAW
Less to set up.
More to change.

No timeline. No session files. mpump is an instrument, not a workspace.
Pick it up, put it down. The groove is always there when you open it.

share
// evolution
A groove is never finished.
It's passed on.

Fork it. Strip the bass. Double the tempo. Keep the kick, rebuild the melody. Every link is a moment in an ongoing mutation. The same groove, reshaped by whoever opens it next.

why
// why I built this

I built mpump to make music anywhere. Not files. Just links and a browser.
Easy to share.

Open it. Hear it. Change it. Send it back different.

Most music tools make you choose: too simple to be useful, or too complex to be fun. This stays in between. Fast to start. Still worth exploring.

No install, no account, no personal data.
Your music stays on your device. Free forever.

Built for techno, acid, and any electronic music that works in loops.
