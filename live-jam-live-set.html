Live Jam / Live Set

Play together live in the browser.

mpump includes two live modes. Live Jam is for a small group shaping the music together. Live Set is for one performer controlling the groove while everyone else listens. In both cases, no audio is streamed. Each browser synthesizes the sound locally inside the same browser groovebox and web electronic instrument.

Two live modes
Live Jam Up to four people can shape the groove together. BPM, effects, mutes, and pad movement stay in sync.
Live Set One controller performs while listeners hear the same performance in their own browsers.
No audio streaming Only control data moves across the relay. Each browser makes its own sound locally, which keeps it lighter and more direct.
Same instrument, live This is not a separate collaboration layer pasted onto the app. It is the same groovebox, opened live.
Why it fits mpump

The live modes are not there to imitate a conferencing product. They fit the same idea as the share links: a groove should stay playable, not collapse into a file too early.
